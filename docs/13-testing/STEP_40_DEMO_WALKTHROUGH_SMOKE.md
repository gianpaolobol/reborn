# Step 40 Demo Walkthrough Smoke Test

`smoke-demo-walkthrough-investor-journey.ps1` validates the guided demo layer:

1. health capabilities include demo mode/walkthrough features;
2. readiness includes `demo_walkthrough`;
3. admin can list demo modes and guided steps;
4. admin can create and advance a demo session;
5. session events are recorded;
6. feedback can be recorded and listed;
7. demo readiness can be evaluated and reviewed;
8. demo audit log records the workflow.

The script is included in `scripts/ci-smoke-tests.ps1` and the Step 39/40 release evidence matrix.
