# MVP Sprint Plan

## Sprint 0 — Repository and product lock

Goal: make the project executable by humans and agents.

Deliverables:

- GitHub authentication and first push;
- PRODUCT.md accepted;
- MVP Lock accepted;
- PRD v0.1 accepted;
- issue templates ready;
- first issues created.

Exit criteria:

- repository is source of truth;
- no local-only strategic documents.
