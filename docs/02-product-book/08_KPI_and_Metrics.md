# Product Book — Capitolo 8: KPI and Metrics

## North Star Metric

**Objects Saved™**: numero di oggetti che tornano funzionanti grazie a Re-born.

## Metriche prodotto

- Repair Cases created
- Repair Cases solved
- Repair success rate
- Time to first repair path
- Time to quote
- Time to completed repair
- AI recognition confidence
- Repair path confidence
- Repeat repair users

## Metriche Knowledge Graph

- Products indexed
- Components indexed
- Failure modes mapped
- Compatibility links created
- Repair DNA records created
- Validation events received
- Knowledge Graph entity growth

## Metriche marketplace

- Maker models uploaded
- Provider registrations
- Active providers
- Quotes generated
- Orders completed
- Gross marketplace volume
- Platform take rate
- Maker royalty generated
- Provider response time

## Metriche trust

- Model repair success rate
- Provider acceptance rate
- Provider delivery reliability
- Dispute rate
- Refund rate
- AI false positive rate
- Review quality score

## Metriche sostenibilità

- Objects Saved
- Estimated waste avoided
- Estimated CO₂ avoided
- Local production ratio
- Average distance saved versus traditional shipping

## Regola

Ogni dashboard deve collegare metriche economiche e metriche di impatto. Re-born non misura solo ricavi: misura vita estesa degli oggetti.
