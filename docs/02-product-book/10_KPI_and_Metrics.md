# 10 — KPI and Metrics

## North Star Metric

**Objects Successfully Repaired**

A repair counts when the user or provider confirms that the object works again or the component fit is successful.

---

## Activation metrics

- repair cases started;
- photo uploads completed;
- object identification success rate;
- component identification success rate;
- percentage of cases with enough data;
- first recommended repair path rate.

---

## Marketplace metrics

- CAD downloads;
- paid downloads;
- print orders;
- provider quote acceptance rate;
- average order value;
- marketplace take rate;
- repeat orders;
- provider response time;
- provider fulfilment success rate.

---

## Maker metrics

- active makers;
- verified models uploaded;
- model success rate;
- maker earnings;
- bounty completion rate;
- model revision frequency;
- model reuse across repair cases.

---

## AI and Knowledge metrics

- recognition confidence distribution;
- correction rate;
- repair path acceptance rate;
- AI-generated model success rate;
- Knowledge Graph nodes added;
- Knowledge Graph relationships added;
- cases linked to existing knowledge;
- learning events per repair.

---

## Sustainability metrics

- objects saved;
- estimated avoided waste;
- repairs by category;
- local production distance saved;
- replacement purchases avoided;
- enterprise circularity reports.

---

## Trust metrics

- dispute rate;
- refund rate;
- failed fit rate;
- provider reliability score;
- model reliability score;
- time to resolution;
- verified repair ratio.
