# Design System Index

## Direzione visiva

Industrial Soft-Futurism / Premium Repair Intelligence.

## Regole

- No Bootstrap.
- No gradient viola.
- No glassmorphism eccessivo.
- Border-radius massimo 4px.
- UI premium, precisa, tecnica.

## Palette iniziale

- Graphite
- Off White
- Repair Green
- Electric Blue
- Safety Orange
