# Frontend Architecture

## Stack

- HTML5;
- CSS3;
- Vanilla JavaScript;
- no frontend framework in the first phase.

---

## Principles

- progressive enhancement;
- semantic HTML;
- accessible forms;
- modular CSS;
- small JavaScript modules;
- no business-critical logic only in browser;
- no heavy UI libraries.

---

## Suggested structure

```text
public/
  index.php
  assets/
    css/
      base.css
      tokens.css
      components.css
      pages.css
    js/
      app.js
      repair-intake.js
      upload.js
      provider-request.js
```
