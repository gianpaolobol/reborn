# MVP Frontend Screen Contracts

## Required global components

- AppShell
- PublicHeader
- AuthForm
- ProgressStepper
- RepairCaseCard
- PhotoUploader
- GuidedQuestion
- DiagnosisCard
- RepairPathCard
- Timeline
- StatusBadge
- EmptyState
- ErrorNotice
- SafetyNotice
- AdminQueueTable
- ClassificationPanel
- QuoteForm
- OutcomeForm

## UI states required for every form

- pristine;
- editing;
- validating;
- saving;
- saved;
- validation error;
- server error.

## Visual constraints

- no heavy glassmorphism;
- no Bootstrap look;
- radius max 4 px;
- graphite/off-white base;
- repair green only for progress/success;
- safety orange only for risk/warnings;
- electric blue only for active action/focus.
