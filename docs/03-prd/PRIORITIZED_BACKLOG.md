## Explicit cut line

If time is limited, do not build full payments, full AI generation or full marketplace browsing.

Instead, build:

- repair case loop;
- provider quote route;
- maker/model metadata route;
- outcome feedback;
- admin learning console.

The MVP must be intelligence-first, not transaction-first.
